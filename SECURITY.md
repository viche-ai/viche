# Security Policy

## Supported Versions

Currently, only the latest version of viche is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of viche seriously. If you discover a security vulnerability within viche, please send an e-mail to Joel Milligan at joel@joeldevelops.com rather than creating a public issue.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If a fix is needed, we will provide a timeline and work to release a patch as quickly as possible.
